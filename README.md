# weather_app

Welcome to the Flutter Weather App, a simple yet effective weather application that demonstrates how to use the GetX package for state management. This app allows users to effortlessly fetch weather data based on their current location or by searching for a specific location, all while providing a smooth and engaging user experience.

## Features

- **Current Location Weather**: The app fetches real-time weather data based on the user's current location, providing up-to-the-minute weather information without the need for manual input.

- **Location Search**: Users can search for weather data in different locations around the world, enabling them to stay informed about weather conditions anywhere they desire.

- **Loading Indicators**: While fetching data, loading indicators ensure that users are aware that the app is actively working to retrieve the information they need.

- **Error Handling**: The app handles errors gracefully and displays informative error messages to users when issues arise during data retrieval.
